# SDM-raytracer

Simple raytracer written in pure C, based on the book [_Ray Tracing in One Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html).

## Compiling and running locally
The following compiles and runs the program, creating the image file `esempio.ppm` in the current directory.

```bash
gcc main.c -o main
main.c > esempio.ppm
```
